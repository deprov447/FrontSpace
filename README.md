Front space is a platform to quickly create single paged websites and host it for the world to see. 

## Build with
Of course love. But also a number of amazing technologies.

### Front end

- React.js
- Chakra UI
- Netlify
- Framer Motion

### Back end

- Express app
- Node.js
- Passport.js
- jsonwebtoken 

### Deployment

- Linode
- Docker
- Docker Compose
- Nginx

### Other

- Gitpod
- Github actions
- Cloudflare
- Hostinger
